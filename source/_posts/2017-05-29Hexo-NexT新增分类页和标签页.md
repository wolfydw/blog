切换到NexT主题后，默认是没有分类页和标签页的，要自己新建

<!--more-->

## 新增菜单

编辑`主题配置文件`

```
menu:
  home: /
  categories: /categories
  about: /about
  archives: /archives
  tags: /tags
  #sitemap: /sitemap.xml
  #commonweal: /404.html
```

##  修改page模版

`\scaffolds\page.md`

```
updated: 
type: "{{ title }}"
comments: false
```



##  新建page

在终端执行

```
hexo new page tags
hexo new page categories
```
